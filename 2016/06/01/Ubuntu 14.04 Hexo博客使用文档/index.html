Ubuntu 14.04 Hexo博客使用文档

Hexo简介

hexo是一个基于Node.js的静态博客程序,可以方便的生成静态网页托管在Github上

使用环境

  • Ubuntu 14.04
  • Node.js v6.0.0
  • Npm 3.8.6

搭建步骤

  • 注册github账号,创建一个yourname.github.io的仓库
  • 买个喜欢的域名(阿里云:https://wanwang.aliyun.com)
  • 下载Node.js、Git(网上教程一大把)
  • 安装Hexo

2.生成ssh密钥

2.1 进入~/.ssh文件夹是否存在
2.2生成 ssh 密钥

3.初始化自己的博客

3.1进入之前创建的blog文件夹,执行命令

deploy:
type: git
repo: https://github.com/ColinFly/ColinFly.github.io.git
branch: master

hexo clean
hexo generate
hexo deploy

hexo new”postName” #新建文章
hexo new page”pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy #将.deploy目录部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本

hexo n == hexo new
hexo g == hexo generate
hexo s == hexo server
hexo d == hexo deploy
